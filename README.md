# Tic Tac Toe Game in C

A console-based two-player Tic Tac Toe game developed using the C programming language.

## Features
- 3x3 game board
- Two-player gameplay
- Input validation
- Win and draw detection
- Clean and modular code structure

## How to Run
1. Compile the program using a C compiler
2. Run the executable in terminal

Example:
gcc tic_tac_toe.c -o tic_tac_toe
./tic_tac_toe
